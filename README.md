# MCustomRecordingStrip
Android 自定义动感歌词文本、自定义录音拖动条

- 自定义动感歌词文本（修改https://github.com/zhangliangming/HPLyrics）完成
- 自定义文本录音拖动条
- 录音拖动条支持手动、自动滑动
- 暴露拖动进度接口给外部调用
- 支持标尺左右不同颜色
- 支持显示时间戳
- 支持颜色、大小、间隔设置
- 具体实现逻辑请看源码MCustomRecordDragView
